## Задание 2
- Установите поддержку LUKS.
- Создайте небольшой раздел, например, 100 Мб.
- Зашифруйте созданный раздел с помощью LUKS.
В качестве ответа пришлите снимки экрана с поэтапным выполнением задания.
## Ответ
Была установлена поддержка LUKS — пакет cryptsetup
![alt text](img/5.JPG)

Дополнительно попыталась подготовить диск 
##### sudo apt install gparted
Поскольку из-за нехватки места (VB) нет технической возможности создать новый раздел, ниже приведены команды, которые необходимо выполнить для шифрования раздела с помощью LUKS.

Шифровать разделы sda1, sda2, sda3 нельзя, так как это может привести к сбоям в работе системы.

sudo cryptsetup -y -v --type luks2 luksFormas /dev/sdb1 — сама шифровка

sudo cryptsetup luksDump /dev/sdb1 — для проверки шифрования
